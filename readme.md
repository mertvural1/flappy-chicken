🐔 Flappy Chicken

Flappy Chicken is a simple and fun 2D browser game inspired by Flappy Bird.
The main goal is to control the chicken and survive as long as possible by avoiding obstacles.

🎮 How to Play
Press Space or Click Mouse to make the chicken jump
Avoid hitting the pipes
Try to get the highest score possible
🚀 Features
🐔 Playable chicken character
🧱 Random pipe generation
📊 Score system
⚡ Simple and addictive gameplay
🌐 Runs directly in the browser
🛠️ Technologies Used
HTML5
CSS3
JavaScript (Canvas or DOM)
📦 How to Run
Download or clone the project:
git clone https://github.com/mertvural1/flappy-chicken.git
Open the folder:
cd flappy-chicken
Open index.html in your browser
🧠 Game Logic
Gravity pulls the chicken down
Player input makes it jump
Pipes spawn randomly
Collision ends the game
Score increases when passing obstacles
